## 思路

用int[100]数组记录每一个成绩的数量即可。

## 代码

[最新代码@github](https://github.com/OliverLew/PAT/blob/master/PATBasic/1038.c)，欢迎交流
```c
{% raw %}#include <stdio.h>

int main()
{
    int N, M, score, count[101] = {0};
    
    scanf("%d", &N);
    for(int i = 0; i < N; i++)
    {
        scanf("%d", &score);
        count[score]++;
    }
    
    scanf("%d", &M);
    for(int i = 0; i < M; i++)
    {
        scanf("%d", &score);
        printf("%d%c", count[score], i == M - 1 ? '\n' : ' ');
    }
    
    return 0;
}
```
