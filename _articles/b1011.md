## 思路

没有难度的题目，32位整型不够用，要用`long int`。

## 代码

[最新代码@github](https://github.com/OliverLew/PAT/blob/master/PATBasic/1011.c)，欢迎交流
```c
{% raw %}#include <stdio.h>

int main()
{
    int T;
    long int A, B, C;
    scanf("%d", &T);
    
    for(int i = 0; i < T; i++)
    {
        scanf("%ld %ld %ld", &A, &B, &C);
        printf("Case #%d: %s\n", i + 1, A + B > C ? "true" : "false");
    }
    
    return 0;
}
```
