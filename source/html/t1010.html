Lehmer Code

According to Wikipedia: "In mathematics and in particular in combinatorics, the Lehmer code is a particular way to encode each possible permutation of a sequence of nnn numbers." To be more specific, for a given permutation of items {A1A_1A​1​​, A2A_2A​2​​, ⋯\cdots⋯, AnA_nA​n​​}, Lehmer code is a sequence of numbers {L1L_1L​1​​, L2L_2L​2​​, ⋯\cdots⋯, LnL_nL​n​​} such that LiL_iL​i​​ is the total number of items from AiA_iA​i​​ to AnA_nA​n​​ which are less than AiA_iA​i​​. For example, given { 24, 35, 12, 1, 56, 23 }, the second Lehmer code L2L_2L​2​​ is 3 since from 35 to 23 there are three items, { 12, 1, 23 }, less than the second item, 35.

Input Specification:

Each input file contains one test case. For each case, the first line gives a positive integer NNN (≤105\le 10^5≤10​5​​). Then NNN distinct numbers are given in the next line.

Output Specification:

For each test case, output in a line the corresponding Lehmer code. The numbers must be separated by exactly one space, and there must be no extra space at the beginning or the end of the line.

Sample Input:

6
24 35 12 1 56 23

Sample Output:

3 3 1 0 1 0
