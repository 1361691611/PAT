Subnumbers (35)

Given a positive integer NNN, let us define a subnumber of NNN as a consecutive number of digits NOT starting with 0. For example if N=1021N = 1021N=1021, it has 7 subnumbers, namely, 1, 10, 102, 1021, 2, 21 and 1 (again). Here is your task: please calculate the sum of all the subnumbers of NNN. For 1021, the sum is 1+10+102+1021+2+21+1 = 1158. Since the result may be very large, output the answer modulo 1000000007 (109+710^9 + 710​9​​+7) please.

Input Specification:

Each input file contains one test case, which gives the integer NNN (0<N<101000000 < N < 10^{100000}0<N<10​100000​​) in a line.

Output Specification:

Print in a line the sum of all NNN's subnumbers (modulo 1000000007).

Sample Input:

1234567890123456789

Sample Output:

332876913
